Membangun Website catalog film

Proses desain situs web dimulai dengan menggunakan pena dan kertas untuk menggambar tata letak halaman, wireframe, peta situs, dan struktur menu. Setelah semuanya selesai lanjut ke tahapan pembuatan tema website dapat dimulai. Proses pengembangan situs web dimulai dengan mengambil elemen-elemen grafis (warna, font, tata letak, gambar) yang ditentukan dalam proses desain dan mengkodekannya menggunakan standar industri web (HTML, CSS, JS). Biasanya hal ini dilakukan dengan pertama kali mengkodekan halaman beranda, kemudian template untuk halaman interior, lalu kumpulkan data film menggunakan Rest API OMDb. Ketika pembuatan website sudah selesai maka harus melalui pengetestan terlebih dahulu setelah selesai, barulah website dapat diluncurkan secara online.


apa yang saya bangun dalam website ini?

Halaman Utama: Ini memiliki fitur paginasi: Jika katalog film memiliki banyak film, saya pertimbangkan untuk membagi hasilnya menjadi halaman-halaman yang dapat dinavigasi. Ini akan memudahkan pengguna untuk menjelajahi katalog film yang saya buat.

Halaman Contact Us: Adalah halaman yang biasanya ada di website yang memungkinkan pengunjung untuk menghubungi saya secara langsung. Halaman ini berfungsi sebagai saluran komunikasi antara pengunjung dan pemilik website, memungkinkan pengunjung untuk mengajukan pertanyaan, memberikan umpan balik, atau mencari dukungan atau bantuan.

Halaman Footer dan Navigasi: Bagian bawah halaman atau footer berisi tautan navigasi tambahan seperti halaman informasi perusahaan, kebijakan privasi, syarat dan ketentuan, dan kontak. Footer juga dapat berisi tautan media sosial, langganan newsletter, atau informasi kontak tambahan.
