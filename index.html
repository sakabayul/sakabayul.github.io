Status

Saya pernah bersekolah di SMK mengambil jurusan Multimedia selama bersekolah saya belajar Arsitektur Informasi, dan karena itu saya memiliki pemahaman yang tajam tentang gambar rangka, peta situs, dan desain. Setelah lulus SMK pada tahun 2019 saya melanjutkan studi untuk masuk sebuah Universitas dan mengambil jurusan Teknik Informasi, di sana saya belajar tentang banyak pemrograman salah satunya adalah tentang Website Developer.

Saya memilih untuk mempelajari Website Desainer karena saya berpengalaman dibidang Multimedia saat bersekolah, yang dimana saya mengerti tentang Arsitektur Informasi, yang saya manfaatkan dengan baik pada setiap proyek yang saya kerjakan. Selama berkuliah saya mendalami sebuah bidang Web Developer dan Desain Multimedia, saya menghabiskan banyak waktu untuk belaja (PHP, HTML, CSS, JS, jQuery, Bootstrap, Responsive Layout, Adobe Illustrator, Adobe Photoshop).

Selama saya berkuliah, saya telah mengerjakan banyak project yang berkaitan dengan Web Developer seperti membuat Design Website E-Commerce, Website Portfolio, Website Pendidikan, Website Portfolio, dan lain-lain. Project tersebut dikerjakan dengan Team Work atau Individual, menggunakan software (VS Code) dengan pengembangan (PHP, HTML, CSS, JS, jQuery, Bootstrap, dan Website Responsif).
